# Page Level UX

Last reviewed: 2026-05-05

## Main page intents
- Dashboard: high-level KPIs and recent activity.
- Demandes/Courriers/Dossiers: operational processing tables.
- Workflow OMA: phase progression and blockers.
- Documents/Reunions/Certificats: specialized follow-up by artifact type.
- Reports: aggregated metrics and distributions.
- Portal preview: external-facing simplified tracking experience.

## Consistent patterns
- Table-heavy admin pages.
- Card-based detail and KPI views.
- Shared loading/error/empty states.
- Read-only informational banners on simulation pages.
